Getting Started
What is jintellv2?

jintellv2 is an all-in-one web library for building application ui/ux focused websites. The targets of the library are single page, portals and desktop applications. The library won't require developers to modify their system structure to use the library. jintellv2 is not a framework and will never be.

Key Features
  • Independent: No-nonsense files or changes to your server.
  • Serverless: No server code required
  • Object-oriented: Javascript over Html.
  • Predefined: Customizable with html over Javascript.
  • Built-in: Built-in many ui controls, demo pages and more.
How to Install jintellv2
  • Add jintellv2 to your project - https://github.com/XHighIntell/jintellv2
  • Add jquery to your project - https://releases.jquery.com/jquery
